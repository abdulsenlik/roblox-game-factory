06
Generate Luau Scripts

~$21
10–50m
ESSENTIAL (combat/economy)
Tools
bulk Claude Sonnet 4.6 Batch (50% off) complex Claude Opus 4.6 cheap Gemini 2.5 Flash
Architecture: Template-First
Pre-built Luau templates for combat, economy, UI, VFX, DataStore. LLM fills parameters (asset IDs, stats, configs) — not writing from scratch. 70% token savings.
Validation Stack
lint Selene (Rust, <1s) format StyLua test Lune runtime
Key Insight
Starts immediately after Stage 1 — only needs GDD manifest, NOT art assets. Runs fully parallel to Stages 2–5, saving 30–40% wall-clock time.
